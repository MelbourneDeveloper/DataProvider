# DataProvider Design System

A clean, technical system for a modern .NET data toolkit. Dark-first, high contrast, no purple.

---

## Brand Principles

- Precise
- Fast
- Trustworthy
- Developer-first
- Minimal ornamentation

You optimize for clarity over decoration.

---
